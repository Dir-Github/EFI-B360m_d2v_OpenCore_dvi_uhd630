# 解锁CFG Lock
```
[GIGABYTE B360m d2v 主板官网](https://www.gigabyte.cn/Motherboard/B360M-D2V-rev-10/sp#sp)

1.首先把这个EFI放在EFI分区下（U盘或者硬盘的都可以，注意备份原来的EFI文件夹）;
2.重启电脑选择从这个EFI所在的盘启动;
3.使用命令: setup_var_3 0x5C1 0x00 即可。0x5C1是我主板bios的CFG Lock设置偏移量, 不保证你的也是这个;
BIOS版本不同可能不一样, 建议先找找教程, 错误使用命令导致的后果自负。
每一个BIOS的设置选项地址（偏移量）都是不一样的，不检查BIOS设置地址而直接使用setup_var命令修改是十分危险的事情！

注意:
如有不明白, 请去百度。
主板型号不一样的话, 可能会无效, 并且可能引发某些不可预料的问题。

请自行百度了解各种风险之后在继续。
```
