# EFI-B360m_d2v_OpenCore_uhd630
OC引导工具 EFI 配置文件

Open Core版本: 0.6.0

# 机器配置
```
CPU: intel i5-8400
内存: 16G
显卡: 核显uhd630
主板: GIGABYTE B360m d2v
声卡: Realtek ALC887
网卡: Realtek 8118

注意自行修改EFI中机型信息
```

# 说明
```
1. 机型为iMac19,1 解决开机鼠标, 键盘会卡顿的现象。
2. 10.15.5 以上核显DVI输出端口可用。
3. 支持全新安装以及OTA升级big sur（macOS 11）。
4. 三码没有填写，请根据自己情况填写，机型设定为iMac19,1
```

# 目前使用体验
```
1. 核显正常驱动
2. 声卡 正常
3. 有线网卡 正常
4. USB 3.0 2.0 正常
5. 睡眠 和 唤醒 待测试，可能会有问题
```
